ML & Databases

My project Saving Sergeant acts as a proof-of-concept for a receipt-scanning and receipt-analyzing software whose aim is to provide financial insights into users' spending habits.

This is done by preprocessing images of receipts, scanning them with Tesseract OCR, and storing them in a csv file that can then be read into a SQL database.
Analysis is then performed with an SQL database to provide insights into the user's spending habits.

See the project here on github.
